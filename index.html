Welcome to Scalus's API

Quick start.

  • Signup for an account at http://go.scalus.com/register
  • Write down the SUBDOMAIN that you created (last field in the form)
  • TODO REGISTER FOR A DEVELOPER ACCOUNT and get your CLIENT_ID & CLIENT_SECRET
  • Connect to the API via oAuth (see ruby example below)
    client = OAuth2::Client.new( CLIENT_ID, 
                                 CLIENT_SECRET, 
                                 site:         'http://SUBDOMAIN.scalus.com', 
                                 organization: 'SUBDOMAIN', 
                                 authorize_url: '/oauth2/authorize',
                                 token_url: '/oauth2/token')
    access_token = client.password.get_token('your_email@address.com', 'YourPassword')
    access_token.token
    # Create a Task:
    
    task = access_token.post("/api/external/tasks", 
                             headers: {'Content-Type' => 'application/vnd.api+json' }, 
                             params:  { data: 
                                        { type: "tasks", 
                                          attributes: { title: 'test 12345', 'due-date' => '2015-12-04'}
                                        }
                                      }
                            )
    
    # Get your first 20 tasks

    task = access_token.get("/api/external/tasks")
